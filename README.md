# Bank Parsing Engine

A Python-based engine for parsing bank statements from various banks, supporting PDF formats.

## Features

- Parse PDF bank statements
- Extract transaction data
- Modular parser architecture for different banks

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`

## Usage

Run the main script: `python main.py`

## Project Structure

- `src/`: Source code
  - `parsers/`: Bank-specific parsers
  - `models/`: Data models
- `tests/`: Unit tests
- `requirements.txt`: Dependencies
